scheleton-mocha-env
======================
Create scheleton of mocha test environment.

Prerequisites
-----
* npm

How to Install
-----
    ./install.sh

Directory ".scheleton-mocha-env" is created under $HOME.

How to Use
-----
    mkdir project-base
    cd project-base
    scheleton-mocha-env

Parameters
----------------
* -d : base directory
    * Default is current directory.
